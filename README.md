🌄 Tenkasi Tourism Website

A simple, informative, and visually appealing tourism website showcasing the beautiful landmarks, waterfalls, temples, and traditions of Tenkasi, Tamil Nadu.

📌 Overview

This project highlights the major tourist attractions in Tenkasi, including:

Courtallam Falls

Kasi Viswanathar Temple

Papanasam

Agasthiyar Falls

Manimuthar Dam

Old Agasthiyar Temple
… and more!

The website is fully static and built using HTML, CSS, and JavaScript (optional).

🚀 Features

🗺️ Clean and user-friendly layout

🖼️ Image gallery of Tenkasi tourist spots

📍 Brief descriptions of each location

📱 Fully responsive design

🎨 Simple and modern UI

🔗 Navigation bar for easy access

🛠 Technologies Used

HTML5 – Structure

CSS3 – Styling & Responsive layout

JavaScript – (Optional) Interactivity, animations
